# Virtusa-Java--Library-Management-System
The Library Management System is a console-based Java application designed to automate basic library operations such as managing books, issuing, and returning them.

This system reduces manual effort and helps maintain accurate records of library transactions.

#  Objectives
Manage book records efficiently

Track issued and returned books

Calculate fines for late returns

Provide quick search functionality

# Concepts Used

Object-Oriented Programming (OOP)

Classes and Objects

ArrayList

Encapsulation

Conditional Statements

Loops and Menu-driven programming
